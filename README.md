# Clone and Run

Para Executar o projeto execute os seguintes passos

- #####  Clone

  Clonar projeto

  ```
  git clone <endereco_do_repositorio>
  ```

- ##### Test web - running by commandline



  ```
  $  mvn clean test -Dsurefire.suiteXmlFiles=src/main/resources/suites/suiteTeste.xml
  ```
